# QueueObj
Queue javascript objects dynamically then process the queue to completion.

Included tag appenders:

* all - process all added objects

Installation
---------
```
npm install QueueObj
```

Mocha Test
---------
```
npm test
```

General Setup Test
---------
```
node test_all
node test_top_one
node test_bottom_one
node test_func_all

```
